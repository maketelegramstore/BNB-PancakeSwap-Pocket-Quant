# BNB-PancakeSwap-Pocket-Quant
Этот проект представляет собой Telegram-бота на базе Google Apps Script, который выступает в роли вашего личного финансового аналитика. Вы можете **голосом или текстом** описать любую торговую стратегию, а бот с помощью ИИ (Gemini &amp; Groq Whisper) сам напишет код, протестирует его на исторических данных и выдаст подробный количественный отчет.
