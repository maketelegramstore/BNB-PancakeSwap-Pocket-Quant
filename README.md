# 🥞 BNB PancakeSwap Pocket Quant 🤖

**Карманный ИИ-квант для 5-минутных раундов PancakeSwap Prediction (BNB)**.

Этот проект представляет собой Telegram-бота на базе Google Apps Script, который выступает в роли вашего личного финансового аналитика. Вы можете **голосом или текстом** описать любую торговую стратегию, а бот с помощью ИИ (Gemini & Groq Whisper) сам напишет код, протестирует его на исторических данных и выдаст подробный количественный отчет.

## ✨ Главные фичи
* 🎙 **Голосовой ввод:** Надиктуйте идею стратегии в Telegram (используется Groq Whisper для транскрибации).
* 🧠 **ИИ-Кодинг:** Gemini AI автоматически переводит ваше текстовое описание в исполняемый JavaScript-код для бэктеста.
* 📊 **Глубокий Бэктест:** Мгновенная симуляция на исторических данных раундов PancakeSwap.
* 📈 **Квантовая аналитика:** Расчет WinRate, PnL, Профит-фактора, просадок (Drawdown), коэффициентов Шарпа и Сортино, а также серийный анализ (Z-Score, Марковские переходы).
* 🔒 **Приватность:** Бот работает строго индивидуально и отвечает только на ваш `CHAT_ID`.

## 🚀 Инструкция по установке и развертыванию (Step-by-Step)

Вам не нужен сервер. Проект работает полностью на бесплатной инфраструктуре Google (Google Sheets + Apps Script).

### Шаг 1. Создание копии базы данных
1. Перейдите по ссылке на шаблон Google Таблицы: **[PancakeSwap Pocket Quant (Шаблон)](https://docs.google.com/spreadsheets/d/1wmmgXLhlj2hRxWWPyx4ZrrU9ix_CcNG4NcTSOcTYCac/copy)**
2. Нажмите синюю кнопку **«Сделать копию»** (Make a copy). *Вместе с таблицей на ваш Google Диск скопируется и весь программный код бота.*

### Шаг 2. Получение API ключей
Для работы бота потребуются 3 ключа и ваш ID в Telegram:
1. **Telegram Bot Token:** Напишите в Telegram боту [@BotFather](https://t.me/BotFather), создайте нового бота командой `/newbot` и скопируйте выданный токен.
2. **Ваш Chat ID:** Напишите боту [@userinfobot](https://t.me/userinfobot) или аналогичному, чтобы узнать свой цифровой ID. Бот будет отвечать *только вам*.
3. **Groq API Key:** Зарегистрируйтесь на [console.groq.com](https://console.groq.com/keys) и создайте бесплатный API ключ (нужен для распознавания голосовых сообщений).
4. **Gemini API Key:** Перейдите в [Google AI Studio](https://aistudio.google.com/app/apikey) и создайте ключ (отвечает за генерацию кода и аналитику).

### Шаг 3. Настройка конфигурации
Откройте вашу скопированную Google Таблицу. Перейдите на лист **`data`**.
Заполните соответствующие ячейки в столбце `B` вашими данными:
* Ячейка `B1` ➔ Токен от BotFather
* Ячейка `B2` ➔ Ваш Telegram ID
* Ячейка `B3` ➔ Ключ от Groq
* Ячейка `B4` ➔ Ключ от Google AI Studio
* Ячейка `B5` ➔ *(Опционально)* Количество знаков после запятой для отчетов (по умолчанию 4).

### Шаг 4. Публикация Webhook (Развертывание скрипта)
Чтобы Telegram мог отправлять сообщения в ваш скрипт, нужно опубликовать его как веб-приложение:
1. В верхнем меню Google Таблицы нажмите `Расширения` ➔ `Apps Script`.
2. В открывшемся редакторе кода в правом верхнем углу нажмите синюю кнопку **Начать развертывание** (Deploy) ➔ **Новое развертывание** (New deployment).
3. Выберите тип: **Веб-приложение** (Web app).
4. Настройки доступа:
   * Выполнять от имени: **Меня** (Me).
   * У кого есть доступ: **У всех** (Anyone).
5. Нажмите **Начать развертывание**. При первом запуске Google попросит предоставить разрешения — со всем соглашайтесь.
6. В конце появится окно с **URL-адресом веб-приложения**. Скопируйте его!

### Шаг 5. Привязка Webhook и Команд
Возвращаемся в Google Таблицу на лист **`data`**:
1. Вставьте скопированный на предыдущем шаге **URL-адрес веб-приложения** в ячейку **`B6`**.
2. Кликните по ссылке в ячейке **`A6`** (`linkApp 🔥 set WEBHOOK`) и перейдите по ней. В браузере должно появиться сообщение от Telegram: `{"ok":true,"result":true,"description":"Webhook was set"}`.
3. Кликните по ссылке в ячейке **`A7`** (`📋 Setup Commands`) и перейдите по ней. Это добавит боту удобное меню команд. В браузере появится `{"ok":true,"result":true}`.

🎉 **Готово! Переходите в своего бота в Telegram и отправляйте ему голосовое или текстовое сообщение с идеей стратегии.**

## 🛠 Пример использования

**Ваш запрос боту:**
> "Давай проверим реверсивную стратегию. Если предыдущий раунд закрылся ростом, ставим на падение. Если падением - ставим на рост."

**Ответ бота (Markdown файл):**
Бот сгенерирует код, прогонит бэктест по встроенной истории и выдаст подробный документ:
* **Винрейт, PnL, Профит-фактор.**
* **Анализ стриков:** Максимальное количество побед/поражений подряд.
* **Математический отчет от ИИ:** Объяснение автокорреляции, вероятностей `P(Win|Win)` и рекомендации по добавлению фильтров (например, не входить в сделку, если кэф ниже 1.85).

## ⚙️ Команды бота
* `/start` - Проверить статус бота.
* `/clear` - Очистить историю диалога (полезно, если ИИ запутался в прошлых версиях стратегии).
* `/history` - Скачать текущую встроенную историю раундов PancakeSwap в формате `.json`.

## ⚠️ Ограничения
* Бот отвечает строго пользователю, чей `CHAT_ID` указан в настройках.
* Суточная квота запросов ограничена бесплатными лимитами Gemini API (если бот выдает ошибку квоты, просто подождите обновления суток).
